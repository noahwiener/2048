2048



Score: 0
How to play: Use your keyboard's arrow keys or your phone/tablet's swipe gestures to slide all of the tiles up, down, left, or right. If two tiles with the same number collide, they combine and double in value! Keep moving and matching tiles until you hit 2048!

This game was created by Noah Wiener using Javascript, jQuery, HTML, and CSS, featuring Matt Bryson's jQuery TouchSwipe plugin for mobile support. Check out some of Noah's other work here. This page is an adaptation of Gabriele Cirulli's original 2048 game.
